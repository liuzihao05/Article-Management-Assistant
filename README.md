此模板应该可以帮助您开始在 Vite 中使用 Vue 3 进行开发。

## 推荐的 IDE 设置

## 自定义配置

参见 [Vite Configuration Reference](https://vite.dev/config/).

## 项目设置

```sh
pnpm install
```

### 用于开发的 Compile 和 Hot-Reload

```sh
pnpm dev
```

### 编译和缩小用于生产

```sh
pnpm build
```

### 使用  [ESLint](https://eslint.org/)的 Lint（弃用）

```sh
pnpm lint
```
